2.memmv和memcpy的区别？手动实现一个memmv

（提出了一个corner case：source_addr=0,target_addr=3,size=5。需要一个额外变量来存储source的值再置0）

```cpp
void memmv(char* source, char* dest, int size) {
	char tmp[size];
	for (int i = 0; i < size; i++) {
		tmp[i] = source[i];
		source[i] = 0;
	}
	for (int i = 0; i < size; i++) {
		dest[i] = tmp[i];
	}
	source = nullptr;
}
```
